# Staff-Leave-Management-System
A comprehensive web-based leave management system built with Python and Django that streamlines employee leave requests, approvals, and tracking.

🌟 Features
Core Functionality
Employee Leave Requests: Submit, edit, and cancel leave applications

Multi-level Approval Workflow: Manager and HR approval processes

Leave Balance Tracking: Automatic calculation of remaining leave days

Multiple Leave Types: Annual, sick, maternity/paternity, compassionate leave

Department-wise Management: Role-based access control

User Management
Role-based Authentication: Employee, Manager, HR Admin

User Profiles: Personal information and leave history

Department Structure: Organizational hierarchy support

Administrative Features
Dashboard Analytics: Leave statistics and reports

Holiday Calendar: Customizable public holidays

Leave Policy Configuration: Flexible policy management

Export Capabilities: Generate reports in PDF/Excel formats

Notifications & Communication
Email Notifications: Automatic status updates

Real-time Updates: Live notification system

Approval Reminders: Automated reminder system

🏗️ System Architecture
Tech Stack
Backend: Django 4.x, Python 3.8+

Database: SQLite-3 (development)

Frontend: HTML5, CSS3, JavaScript, Bootstrap 5

Authentication: Django Allauth

API: Django REST Framework (optional)

Prerequisites
